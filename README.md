Stack technologies:
-------------------
* React 15
* Webpack 2
* Redux
* HMR 3
* Promises
* ESLint
* NPM

Commands
--------
* Install project: npm i
* Start project: npm run dev (starting in: http://localhost:8080)
* Build project: npm run build
